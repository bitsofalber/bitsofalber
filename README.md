## Featured — KrakenSOC-Labs

> **Complete Blue Team labs, packaged in Docker.** You download a lab, spin it up on your own
> machine and investigate real attacker techniques — with real tools (Wireshark, tcpdump, **a real Splunk**).
> Every lab is deterministic, MITRE ATT&CK–mapped, and ships Sigma/Suricata detections.
